### أوامر التثبيت لنظام Ubuntu , Linux Mint , Debian [ terminal ]


```bash
   > sudo apt-get install nodejs
   > sudo apt-get install git
   > git clone https://github.com/rn0x/Bot_Telegram_WebSite
   > cd Bot_Telegram_WebSite
   > npm i
   > npm start
```
